# Code-de-mon-PFE
Code Python du PFE Master en Actuariat &amp; Finance des marches. Modélisation comportementale des dépôts à vue et son impact sur la gestion des risques du liquidité. Comparaison Holt-Winters vs XGBoost. Préparation des données, entraînement des modèles, évaluation (MAE, RMSE, MAPE,R²) et convention d'écoulement. Réalisé sous Google Colab.
